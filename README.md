# BalanceBuddy v2

Modern expense sharing application built with React, Express, PostgreSQL and Docker.

## Tech Stack

- React 19
- TypeScript
- Vite
- Express
- Prisma
- PostgreSQL
- Redis
- Socket.IO
- Docker
